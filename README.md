# AI Resume Screening System 🚀

## 📌 Project Overview
The AI Resume Screening System is a web-based application designed to automate the process of analyzing resumes. It extracts key skills, evaluates candidates based on predefined criteria, and provides intelligent insights to assist recruiters.

This system also includes a chatbot, login system, and admin dashboard, making it a complete full-stack AI-based project.

---

## 🎯 Objectives
- Automate resume screening process
- Reduce manual effort in recruitment
- Provide skill-based scoring
- Enhance hiring efficiency using AI

## ⚙️ How It Works
1. User logs into the system
2. Uploads resume file
3. System extracts text from resume
4. AI analyzes skills and assigns score
5. Results are displayed to user
6. Admin can view all users
